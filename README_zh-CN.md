## 简介
OpenClaw Manager 是 OpenCLAW 的图形化管理工具，解决部署复杂、成本失控、安全风险等问题。

## 功能特性
- ✅ 图形化配置工具（支持模板）
- ✅ Token 成本监控与预警
- ✅ 多实例管理
- ✅ 密钥加密存储
- ✅ 安全体检与加固
- ✅ Skills 管理与扫描
- ✅ 错误中文解释
