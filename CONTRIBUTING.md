# 贡献指南

感谢你为 OpenCLAW Manager 做出贡献！

## 📋 如何贡献

### 1. 报告 Bug

发现 Bug？请提交 Issue：
- 使用 **[Bug 报告模板](.github/ISSUE_TEMPLATE/bug_report.md)**
- 提供清晰的复现步骤
- 附上截图或日志

### 2. 提出功能建议

有好的想法？欢迎提建议：
- 使用 **[功能建议模板](.github/ISSUE_TEMPLATE/feature_request.md)**
- 说明功能的动机和用途
- 描述期望的行为

### 3. 提交代码

想直接贡献代码？
```bash
# 1. Fork 仓库
# 2. 克隆到本地
git clone https://github.com/YOUR_USERNAME/ocm-manager.git
cd ocm-manager

# 3. 创建分支
git checkout -b feature/your-feature-name

# 4. 开发并提交
git commit -m "feat: add your feature"

# 5. 推送到远程
git push origin feature/your-feature-name

# 6. 创建 Pull Request
```

## 🎯 开发流程

### 环境搭建
```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm tauri dev

# 运行测试
pnpm test
```

### 代码规范
- 遵循现有的代码风格
- 添加必要的注释
- 确保编译通过

### 提交信息规范
```
feat: 新功能
fix: Bug 修复
docs: 文档更新
style: 代码格式调整
refactor: 重构
test: 测试相关
chore: 构建/工具相关
```
