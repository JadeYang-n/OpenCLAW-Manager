### 方式二：手动操作

**步骤**:
1. 访问 https://github.com/new
2. 创建仓库 `ocm-manager`
3. 复制仓库地址
4. 执行命令：
   ```bash
   git remote add origin https://github.com/你的用户名/ocm-manager.git
   ```
